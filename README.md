# 🛒 BD Kamer Shop

## 📝 Description
**BD Kamer Shop** est une solution de commerce électronique complète conçue pour digitaliser les ventes et la gestion de stock. Le projet vise à offrir une expérience d'achat fluide et sécurisée adaptée aux besoins locaux.

## ✨ Fonctionnalités
* **Catalogue Dynamique :** Affichage des produits par catégories.
* **Gestion du Panier :** Ajout, modification et suppression d'articles.
